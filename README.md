# Despliegue Automático en Azure

#
Nombre: SHEYLA AQUINO
Curso: INGENIERIA DE SOFTWARE

# consiste en una aplicación web sencilla desarrollada con Node.js y Express, desplegada automáticamente en Azure mediante integración continua 

# Tecnologías
- Node.js
- Express
- Azure 
- GitHub 

# Funcionamiento
Cada vez que se realiza un push al repositorio en GitHub, se ejecuta un pipeline automático que despliega la aplicación en Azure.

# Resultado
La aplicación muestra un mensaje de bienvenida en el navegador
